Directory content:
- cheetah_repro.zip: a cheetah repro set for several orgs, from ABLE data
- shorenstein_repro/: the harvard shorenstein web-content 2016 dataset appended with cheetah scores, gzipped and split into multiple fragments
                      as output by GzSplitter in src/util/file_splitter. Use GzSplitter.Unsplit() to unsplit and uncompress to csv.
